# Temperature Trend Analysis — São Paulo (INMET)

This project explores temperature data from the INMET meteorological station located in São Paulo (Interlagos).

The objective is to analyze temperature patterns and estimate a linear trend in average annual temperature.

## Data Source

Data was obtained from the Brazilian National Institute of Meteorology (INMET).

Meteorological Station:
São Paulo – Interlagos

Period analyzed:
2020 – 2025

## Objectives

- Load and process meteorological data from multiple CSV files
- Clean and organize the dataset
- Calculate daily, monthly, and annual average temperatures
- Compute temperature anomalies
- Estimate a linear temperature trend using regression
- Visualize the results

## Tools Used

Python

Libraries:
- Pandas
- Matplotlib
- SciPy
- Jupyter Notebook

## Analysis Steps

1. Data loading from INMET CSV files
2. Data cleaning and datetime creation
3. Aggregation of daily temperature data
4. Calculation of monthly and annual averages
5. Temperature anomaly analysis
6. Linear regression to estimate temperature trend
7. Visualization of results

## Example Visualization

The analysis includes plots of temperature evolution and a linear trend estimation.

## Results

The linear regression suggests a slight increase in average temperature during the analyzed period.

However, the short time span (2020–2025) does not allow strong conclusions about long-term climate change.
